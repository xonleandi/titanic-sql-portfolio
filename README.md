# 🚢 Titanic Database Analysis Portfolio

## 📊 О проекте
Анализ данных пассажиров Титаника с использованием PostgreSQL и Python.  
Проект демонстрирует навыки администрирования баз данных, написания сложных SQL-запросов и визуализации данных.

## 🛠️ Технологии
| Технология | Назначение |
|------------|------------|
| **PostgreSQL** | Хранение и анализ данных |
| **SQL** | Написание запросов, создание представлений (VIEW) |
| **Python (pandas, matplotlib)** | Визуализация данных, автоматизация |
| **pgAdmin** | Управление базой данных |
| **Git / GitHub** | Контроль версий и портфолио |

## 🚀 Как запустить проект

### 1. Установите PostgreSQL
- Скачайте с [postgresql.org](https://www.postgresql.org/download/)
- При установке запомните пароль (в проекте используется `postgres`)

### 2. Создайте базу данных и загрузите данные
- Откройте pgAdmin
- Создайте базу `titanic_db`
- Импортируйте датасет Titanic (CSV) в таблицу `titanic`

### 3. Выполните SQL-запросы
- Откройте файлы из папки `queries/` в Query Tool
- Выполните их по очереди

### 4. Установите Python-библиотеки
